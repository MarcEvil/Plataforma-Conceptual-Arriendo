# Hogar. | Experiencias de Alojamiento en Santiago

Este proyecto es una plataforma web moderna y minimalista diseñada para la gestión de arriendos y experiencias turísticas en Santiago de Chile. Se enfoca en una interfaz de usuario (UI) de alto impacto, utilizando efectos visuales avanzados y un diseño orientado al usuario.

## 🚀 Características Principales

*   **Interfaz Aura Dinámica:** Implementación de un resplandor radial tricolor que sigue el movimiento del cursor en tiempo real mediante variables CSS y JavaScript.
*   **Diseño Responsive:** Totalmente adaptado para dispositivos móviles, tablets y escritorio utilizando **Tailwind CSS**.
*   **Experiencia de Usuario (UX):** Navegación fluida entre secciones de exploración, perfiles de anfitriones y catálogo de experiencias locales.
*   **Micro-interacciones:** Animaciones de entrada en cascada (stagger effect) y transiciones suaves de elevación en tarjetas.

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** HTML5, Tailwind CSS (CDN), CSS3 (Custom Variables & Pseudo-elements).
*   **Interactividad:** JavaScript Vanilla (ES6+) para manipulación del DOM y lógica de carrusel.
*   **Iconografía:** Font Awesome 6.0.
*   **Tipografía:** Inter / Sistema de fuentes Sans-serif modernas.
